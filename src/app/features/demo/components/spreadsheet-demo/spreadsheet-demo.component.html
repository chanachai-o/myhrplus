Spreadsheet Component Demo

A comprehensive spreadsheet component with Excel-like functionality including formulas, formatting, charts, and more.

Basic Usage 

// TypeScript
import {{ '{' }} SpreadsheetComponent {{ '}' }} from '../../shared/components/spreadsheet/spreadsheet.component';

@ViewChild('spreadsheet') spreadsheet!: SpreadsheetComponent;

sheets = [
  {{ '{' }}
    name: 'Sheet1',
    ranges: [{{ '{' }} dataSource: dataArray {{ '}' }}]
  {{ '}' }}
];

// HTML
<app-spreadsheet
  #spreadsheet
  [sheets]="sheets"
  [height]="'600px'"
  [width]="'100%'">
</app-spreadsheet>

Actions

Features

  • Excel-like editing and formatting
  • Formula support (SUM, AVERAGE, COUNT, etc.)
  • Charts and graphs
  • Conditional formatting
  • Data validation
  • Cell merging and freezing
  • Sorting and filtering
  • Import/Export (Excel, CSV, PDF)
  • Undo/Redo functionality
  • Multiple sheets support
  • Hyperlinks and images
  • Find and replace
